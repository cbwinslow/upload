# CloudCurio Infrastructure

This directory contains infrastructure as code and deployment configurations.

## Infrastructure Components

### terraform
Terraform configurations for cloud infrastructure deployment.

### ansible
Ansible playbooks for server configuration and management.

### docker
Docker configurations and compose files for services.

## Environment Setup

Infrastructure is organized by environment:
- `development/` - Development environment configs
- `staging/` - Staging environment configs  
- `production/` - Production environment configs

## Deployment

See individual directories for deployment instructions.
