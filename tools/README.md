# CloudCurio Tools

This directory contains utility tools and automation scripts.

## Tool Categories

### ai-automation
AI automation frameworks and tools for various platforms:
- Linear automation
- GitHub tools  
- GitLab integration
- Jira automation
- MCP autowire
- Mega installer

### ai-doc-framework
Documentation framework for AI projects with templates and scripts.

### gemini-cli
Gemini CLI tools and extensions for AI development.

### openai-agents
OpenAI agents package and CrewAI bridge tools.

### automation
General automation tools and configuration files.

### github-sync
GitHub repository synchronization and management tools.

### networking
Network analysis and API tools.

## Usage

Each tool directory contains its own README with setup instructions.
