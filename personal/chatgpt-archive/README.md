# ChatGPT Archive

This directory contains your complete ChatGPT conversation history export.

## Contents

- `conversations.json` - Full conversation history with metadata
- `user.json` - User profile and account information  
- `chat.html` - Web-viewable conversation archive
- `group_chats.json` - Group chat conversations
- `shared_conversations.json` - Shared conversation data
- `message_feedback.json` - User feedback on responses
- `shopping.json` - Shopping-related conversations
- `dalle-generations/` - DALL-E generated images
- Various image files and conversation assets

## Security Notice

This folder contains sensitive personal data including:
- Complete conversation history
- Personal information in user profile
- Potentially sensitive discussions

**Do not commit to public repositories or share publicly.**

## Usage

- Use `chat.html` to browse conversations in a web interface
- JSON files can be analyzed for insights or patterns
- Consider setting up proper backup and encryption for long-term storage

## Processing Ideas

- Extract useful prompts and patterns for reference
- Analyze conversation topics and trends
- Create personal knowledge base from insights
- Set up automated backup to secure location
